# Firebase Architect - Template Preview Web App

A standalone web application for browsing, customizing, and selecting UI templates for Firebase projects.

## Features

- **Template Gallery**: Browse 6 professional UI templates
- **Live Preview**: See how templates look with your customizations
- **Color Customization**: Adjust primary, secondary, and accent colors
- **Font Selection**: Choose from 6 pre-configured font combinations
- **Quick Presets**: Apply color palettes instantly
- **Code Generator**: Get ready-to-use configuration code

## Getting Started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

## Templates Included

1. Material Modern - Google Material Design
2. Ant Design Pro - Enterprise admin interfaces
3. TailAdmin Modern - Minimalist Tailwind design
4. CoreUI Enterprise - Bootstrap-based corporate UI
5. Shadcn Modern - Radix UI + Tailwind
6. Clean Accessible - WCAG AAA compliant design
